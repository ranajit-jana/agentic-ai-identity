# 🛠️ Build Just ONE Thing (start here)

## “Secure Tool Call”

Flow:

1. Agent gets identity (JWT or SPIFFE ID)
2. Agent calls tool API
3. API checks:

   * token valid?
   * allowed by OPA?
4. allow / deny

---

# 🧪 Minimal Project Structure

* `agent.py` → makes request with token
* `gateway.py` → verifies token
* `policy.rego` → rules (OPA)
* `tool_api.py` → protected tool

---

# 🧠 What to Learn (only 3 things)

1. JWT signing & validation
2. OPA policies (Rego basics)
3. Service-to-service auth (SPIFFE optional)
